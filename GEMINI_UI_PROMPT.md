# Gemini UI Design Prompts (Copy & Paste)

## Prompt 1: Settings Page Design
```
Generate a high-fidelity iOS UI design mockup for a Settings screen in dark mode, ROG gaming style with red and cyan neon accents. The design should include:

Layout:
- Dark background (#121212)
- Section headers with red neon borders (#FF3333)
- Setting rows with cyan icons (#00CCFF)
- Toggle switches with neon glow effects
- List items with subtle scanline texture

Sections to include:
1. Test Configuration (picker rows)
2. Data & Privacy (toggles, export/clear options)
3. Appearance (dark/light mode selector)
4. About (version info, links)
5. Advanced (sensor tools, debug info)

Style: Cyberpunk gaming UI, iOS 17+, high contrast, tech aesthetic
Output: Image with 1280x2772 resolution, showing the full settings screen
```

## Prompt 2: Sustained Gaming Test View Design
```
Generate a high-fidelity iOS UI design mockup for a gaming performance test screen, ROG style with red and cyan neon accents. Dark mode, iPhone 15 Pro dimensions.

Layout:
- Dark background (#0E0E10) with subtle circuit board overlay
- Large circular gauge at center showing "Stability" percentage in red neon (#FF3333)
- Status bar at top with cancel button
- Grid of real-time stats cards (CPU, Memory, GPU, Latency, Jitter, Packet Loss)
- Progress bar at bottom
- Main action button (Interrupt/Complete) in red gradient

Style: Sci-fi gaming HUD, scanlines, neon glow borders, futuristic data visualization
Output: Image with 1280x2772 resolution, full-screen immersive design
```

---

## How to use:
1. Go to https://gemini.google.com
2. Copy the entire prompt (including the "Generate..." line)
3. Paste into Gemini
4. Gemini will generate the UI mockup image

## Tips for better results:
- Use English prompts (Gemini works best with English)
- Be specific about colors, layout, and style
- Mention "iOS" and "dark mode" explicitly
- Ask for "high-fidelity mockup" or "UI design"
